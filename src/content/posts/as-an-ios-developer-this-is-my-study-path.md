### <a name="tools"></a> Tools
[Reveal](http://revealapp.com/)  
Reveal brings powerful runtime view debugging to iOS developers

[mogenerator](http://rentzsch.github.io/mogenerator/)  
Generates Objective-C code for your Core Data custom classes

[synx](https://github.com/venmo/synx)  
A command-line tool that reorganizes your Xcode project folder to match your Xcode groups

[oh my zsh](http://ohmyz.sh/)  
Your terminal never felt this good before

[Objective-Clean](http://objclean.com/index.php)  
Coding style tool

[Postman](https://www.getpostman.com/)  
Build, test, and document your APIs faster

### <a name="xcode-plugins"></a> Xcode plugins
[Alcatraz](http://alcatraz.io/)  
The package manager for Xcode

[XVim](https://github.com/XVimProject/XVim)  
Xcode plugin for Vim keybindings 
(You can install it from Alcatraz)

[XCoverage](https://github.com/dropbox/XCoverage)  
Displays test coverage data in editor
(You can install it from Alcatraz)

[XcodeColors](https://github.com/robbiehanson/XcodeColors)
Allows you to use colors in debugging console
(You can install it from Alcatraz)

[VVDocumenter-Xcode](https://github.com/onevcat/VVDocumenter-Xcode)  
Helps you write Javadoc style documents easier
(You can install it from Alcatraz)

[KSImageNamed](https://github.com/ksuther/KSImageNamed-Xcode)  
Provides autocomplete for `imageName:` calls
(You can install it from Alcatraz)

[FuzzyAutocomplete](https://github.com/FuzzyAutocomplete/FuzzyAutocompletePlugin) A Xcode 5+ plugin that adds more flexible autocompletion rather than just prefix-matching.

[ShowInGitHub](https://github.com/larsxschneider/ShowInGitHub) Xcode plugin to open the GitHub page of the commit of the currently selected line in the editor window.
